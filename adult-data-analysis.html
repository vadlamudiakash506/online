Adult Data Analysis

Project Overview

This dashboard analyzes adult income data to identify salary trends based on education, occupation, working hours, gender, and demographics. The project helps users understand how different factors influence income distribution through interactive charts and KPI cards.

Key Features

  • ✔ Interactive KPI Dashboard
  • ✔ Education vs Income Analysis
  • ✔ Occupation Insights
  • ✔ Gender-Based Comparisons
  • ✔ Dynamic Filters & Slicers
  • ✔ Clean Dashboard Design
