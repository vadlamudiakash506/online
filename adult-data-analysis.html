Adult Data Analysis

Excel • Power Query • Dashboard Design

Project Overview

Excited to share my latest Excel Data Analytics project — an interactive Adult Data Dashboard! This dashboard delivers a comprehensive analysis of adult demographics, occupations, education levels, and income distribution using powerful Excel visualization techniques. Key Insights from the Analysis: Total Records Analyzed: 32,561+ Gender Distribution: • Male: 21,790 • Female: 10,771 Marital Status vs. Working Hours Married individuals were found to work higher average hours compared to other groups. Education vs. Income Higher education levels showed a strong correlation with salaries above $50K. Occupation Trends Executive managers and professional roles dominated the higher-income category. Age & Education Analysis Interesting patterns emerged between average age and education levels. This project demonstrates how data storytelling and interactive dashboards in Excel can transform raw datasets into meaningful business insights. Tools Used: • Microsoft Excel • Pivot Tables • Charts & Visualizations • Slicers & Interactive Filters • Data Cleaning & Analysis Data analytics is not just about numbers — it’s about uncovering stories hidden within the data.
