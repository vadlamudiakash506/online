Breast Cancer Analysis

Excel • Power Query • Dashboard Design

Project Overview

Turning Data into Insights: Breast Cancer Analysis Dashboard Every dataset tells a story. This time, I worked on a Breast Cancer Analysis Dashboard in Power BI to transform healthcare data into meaningful insights that can support awareness and decision-making. What I Built: Interactive 2-page dashboard KPI Cards: Total Patients, Avg Age, Avg Tumor Size, Confirmed Cases Cancer Type comparison visuals Survival Months by Cancer Stage Recurrence Analysis by Cancer Type Gender-based insights with slicers & filters Key Findings: Female patients accounted for the majority of reported cases Ductal Carcinoma had the highest number of patients Triple Negative cases showed higher average tumor size Early detection stages had stronger survival outcomes Skills Applied: Power BI | Power Query | DAX | Data Cleaning | Data Storytelling | Healthcare Analytics This project reminded me that analytics is not just about charts—it’s about creating impact through insights.
