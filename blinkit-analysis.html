Project Overview

New Excel Project Alert: Blinkit Sales Analysis I'm thrilled to share my latest data visualization project using Excel! Project Title: Blinkit Sales Analysis Objective: To analyze sales performance for Blinkit, including total sales, item categories, outlet performance by size and location, fat content trends, and yearly sales patterns. Key Insights Visualized: Total Sales: $1.2M Total Items Sold: 8523, Avg Sales: $141, Avg Rating: 4.0 Top Item Types: Fruits & Vegetables (178.1K), Snacks (176.4K), Household (136.0K) Sales by Outlet Size: Tier 3 (427.1K), Tier 2 (399.2K), Tier 1 (346.0K) Yearly Sales Trend: Peak in 2019 (204.5K), decline post-2020 Fat Content Distribution: Low Fat (425.4K), Regular (50.4K) Visual Filters for Outlet Size, Location, and Item Type This dashboard integrates KPIs, histograms, pie charts, bar graphs, and area charts to tell a compelling sales story. I used filters and color coding to enhance interactivity and insight discovery. Tools Used: Excel Data Cleaning & Transformation
